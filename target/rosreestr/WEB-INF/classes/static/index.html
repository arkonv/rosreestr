Описание полей

  • address - Адрес (местоположение)
  • pubdate - Дата опубликования на ПКК (Публичная кадастровая карта)
  • date_cost - Дата внесения кадастровой стоимости
  • fp - Форма собственности
  • Кадастровая стоимость, собирается из двух полей, cad_cost и cad_unit, где cad_unit - код единицы измерения стоимости
  • Декларированная площадь - собирается из трех полей - area_value, area_type и area_unit, где, area_type - тип площади , area_unit - код единицы измерения площади
  • Разрешенное использование: util_code - код, util_name - описание, util_by_doc - список разрешенных использований по классификатору (см. раздел 3.3) по документу
  • date_create - Дата постановки на учет
  • cad_record_date - Дата изменения сведений в ГКН
  • rights_reg - Признак того, что на объект недвижимости зарегистрированы в ЕГРП действующие права: 0 - нет прав, 1 - есть права
  • statecd - Статус

Кадастровая организация

  • cad_eng_data - Кадастровый инженер
  • actual_date - "Tue, 19 Jul 2016 00:00:00 GMT"
  • co_name - "ООО Техническое Бюро Кадастра"
  • lastmodified - "Tue, 19 Jul 2016 19:21:39 GMT",

Пример:

"cad_eng_data": {
"actual_date": "Mon, 24 Apr 2017 00:00:00 GMT",
"co_name": "ООО \"ГИПРОЗЕМ\"",
"lastmodified": "Mon, 24 Apr 2017 20:46:20 GMT"
}

Кадастровые номера

  • cn - кадастровый номер земельного участка (ЗУ)
  • kvartal_cn - кадастровый номер квартала (например, городского квартала)
  • rayon_cn - кадастровый номер района (как правило район области)
  • okrug_cn - кадастровый номер округа (т. е. области, края и т. д.)

Примеры оформления

  • Справочная информация по объектам недвижимости в режиме online
  • Публичная кадастровая карта
